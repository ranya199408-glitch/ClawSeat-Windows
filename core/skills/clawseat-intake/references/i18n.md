# i18n

## Mirror Language

Respond in the user's language. If the user mixes languages, mirror the structure of the dominant language and keep technical nouns as written.

## Mixed Chinese/English

Use Chinese structural words with technical nouns preserved:

```text
PR 已通过 CI，可以 merge。
```

Avoid awkward literal mixing:

```text
PR has 通过了 CI，可以 merged。
```

## Glossary Lookup Order

1. Project glossary: `~/.agents/memory/projects/<project>/glossary-project.toml` or the project's local `glossary.toml`.
2. Shared base: `references/glossary-global.toml`.
3. `shared-tone.md` tier table.
4. LLM judgment, with the user's current wording as the tie-breaker.

Project glossary entries override the global table. If no glossary exists, use the tier table and avoid inventing private vocabulary.
