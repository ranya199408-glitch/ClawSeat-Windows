# clawseat-ancestor (alias backstop)

This directory is an alias backstop for the canonical `clawseat-memory` skill.

The `SKILL.md` here declares:

- `name: clawseat-memory`
- `aliases: [clawseat-ancestor]`

**Canonical location**: `core/skills/clawseat-memory/SKILL.md`

This directory exists so that skill loaders referencing the old
`clawseat-ancestor` name continue to work during the alias compatibility period
(until 2026-10-28). After sunset, this directory may be removed.
